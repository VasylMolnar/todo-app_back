# 📝 Todo Fullstack App (Next.js + Node.js + SQLite)

A simple full-stack Todo application with categories, built as a test task.

## 🚀 Tech Stack

### Frontend

- Next.js (React)
- TypeScript
- Mantine Form
- React Toastify
- Axios
- TailwindCSS

### Backend

- Node.js
- Express.js
- SQLite

---

# Frontend Setup

npm install
npm run dev

Create .env.local: NEXT_PUBLIC_API_URL=http://localhost:3001/api

# Backend Setup

npm install
npm start
